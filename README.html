First Download the .zip file than extract it by doing a two-finger click while on the file. Once the file is extracted, it should create a folder called Soundboard V1, or whatever version you downloaded. Inside that folder will be a folder called SoundboardData.




Inside that folder will be the .HTML file that you need to launch. That HTML file contains the actual soundboard.







Once you open the file click on the button that is labeled Pick Master Folder.








Then find the folder named SoundboardDate then click open





Then hit allow on the permissions and then it will load the default sound file. (There will be a feature for multiple sound files)
